Fixing Git Mistakes

  1. Rename last commit
    • let's commit: "git commit -m "Add index.htmll"
    • Now we want to change the commit message: git commit --amend -m "Add index.html"
    • Check last commit log: "git log --oneline"

  2. Add new changes to last commit
    • Make changes and add untracked file : "git add -A"
    • Add new changes to last commit: git commit --amend -m "Adding index.html and app.js"
    • Check the last commit: "git log --oneline"

  3. Git remove file from staging before commiting
    • Let's add new file: "touch lib.js"
    • Add this file to staging area: "git add lib.js"
    • Check the status of file "git status" and check the head position by "git log --oneline"
      • Ex: 6cdfc0e (HEAD -> master) Adding index.html and app.js
        f86d914 (origin/master, origin/HEAD) Initial commi"
    • Now reset head : "git reset HEAD lib.js"
    • Check the status and remove the file.
